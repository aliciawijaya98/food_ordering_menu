# Food Ordering Application
This project was developed as part of the **Purwadhika Data Science & Machine Learning** bootcamp. It is a Python-based CLI food ordering mini-application that allows users to register and log in, browse restaurant menus, manage shopping carts, and place orders using CRUD operations, built entirely with Pythonâ€™s standard library.

# Features
1. User registration and login
2. Choose order type: dine-in, takeaway, or delivery
3. Browse restaurant menu
4. Add items to cart
5. Update or remove items from cart
6. Checkout and place an order
7. Delivery address input for delivery orders
